# Building on Ubuntu 22.04 with CUDA 12.6

These instructions are for Ubuntu 22.04 with CUDA 12.6

* Step 1. Build and install OpenCV.

* Step 2. You can now create makefiles, which will allow you to
build and run your homework on your own machine:
```
mkdir build
cd build
cmake ..
make
```
